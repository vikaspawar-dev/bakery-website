# 🍰 Cakery - Bakery Website

A modern and responsive bakery website built using **HTML, CSS, and JavaScript**.
This project showcases a clean UI design, smooth scrolling, and mobile-friendly layout.

---

## 🚀 Live Demo

🔗 https://vikaspawar-dev.github.io/bakery-website/

---

## ✨ Features

* Responsive design (Mobile + Desktop)
* Smooth scrolling navigation
* Modern UI layout
* Interactive cards & hover effects
* Mobile hamburger menu
* Clean and structured code

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
